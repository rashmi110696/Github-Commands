- Initialise the git
`git init`

- check branch
`git branch`

- create a new branch and go to that branch
`git checkout -b "branch_name"`

- check the status of the file
`git status`

- Add the file
`git add file_name`

- save the added file (commit the file)
`git commit -m "commit_message"`

- push the commited file to remote github
`git push origin branch_name`

- check diff of a file
`git diff`

- to update the recent status to master branch, go to master and do git pull
`git pull`
